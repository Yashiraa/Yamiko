# Yamiko
A simple template using the Discord library.

## How to install ?

You need to have Node.JS 16+
``````bash
git clone https://github.com/Yashiraa/Yamiko
cd Yamiko
npm i
``````

## How to install dependencies ?
```bash
npm i discord.js dotenv
npm init -y
```
